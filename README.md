# DREAM ON — Web RPG Prototype

Github Pages에 바로 올릴 수 있는 DREAM ON 웹 RPG 프로토타입입니다.

## 포함 기능
- 직업 3종 선택
- 마을 / 사냥터 / 전투 / 상점
- 레벨업 및 스탯 분배
- 꿈 파편 + Lucid / Nightfall 성향
- 소프트캡 / 하드캡 기반 퍼센트형 스탯 제한
- LocalStorage 저장

## 바로 실행하기
1. 이 폴더를 그대로 GitHub 저장소에 업로드
2. 저장소 Settings → Pages
3. Branch를 `main` / root로 설정
4. 배포 주소 접속

## 파일 구조
- `index.html` : 메인 화면
- `style.css` : UI 스타일
- `main.js` : 게임 로직
- `data/gameData.js` : 클래스, 몬스터, 지역, 상점 데이터

## 밸런스 메모
초기 밸런스 규칙은 업로드된 기획 메모를 반영했습니다.

- 상태이상 저항: 소프트캡 40 / 하드캡 75
- 보스 상태이상 저항: 소프트캡 30 / 하드캡 50
- 회피율: 소프트캡 20 / 하드캡 45
- 치명타 확률: 소프트캡 25 / 하드캡 50
- 공격속도: 소프트캡 20 / 하드캡 50
- 쿨타임 감소: 소프트캡 15 / 하드캡 35
- 피해 감소율: 소프트캡 20 / 하드캡 50
- 하드캡의 60% 이후 효율 반감 방향 유지
