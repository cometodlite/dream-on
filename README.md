# DREAM ON Graphic Prototype v0.3

Phaser.js 기반의 그래픽 프로토타입이다.
Github Pages에 바로 업로드 가능한 정적 웹 구조로 작성했다.

## 포함 기능
- 타이틀 화면
- 직업 선택: 몽검사 / 성흔사제 / 악몽술사
- 여명 성채 이동형 허브 맵
- 안개의 숲 이동형 필드 맵
- 몬스터 접촉 후 전투 씬 진입
- Lucid / Nightfall 표시
- EXP / 골드 / Dream Shard 보상
- LocalStorage 저장 / 불러오기

## 조작
- 이동: WASD 또는 방향키
- 상호작용: E
- 전투: 1 / 2 / 3 / 4
- 전투 종료 후: 스페이스바

## 배포 방법
1. 이 폴더를 Github 저장소에 업로드
2. `Settings > Pages` 에서 브랜치 선택
3. 루트(`/`)를 배포 대상으로 지정
4. 몇 분 후 `index.html` 기준으로 실행 가능

## 주의
- 현재 버전은 외부 이미지 없이 도형과 색상으로만 구성한 최소 프로토타입이다.
- 이후 `assets/` 폴더에 실제 스프라이트, 타일셋, UI 리소스를 교체 연결하면 된다.
- CDN으로 Phaser를 불러오므로 인터넷 연결 환경에서 실행하는 것이 안정적이다.
