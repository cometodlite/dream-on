# DREAM ON - Text Web RPG

Github Pages에 바로 올릴 수 있는 정적 웹 RPG 프로토타입입니다.

## 포함 기능
- 직업 선택 3종: 몽검사 / 성흔사제 / 악몽술사
- 지역: 여명 성채 / 안개의 숲 / 침묵 수도원
- 텍스트 전투
- 레벨업 + 스탯 분배
- 상점 / 장비 / 인벤토리
- Lucid / Nightfall / 꿈 파편
- 퀘스트 2종
- LocalStorage 저장 / 불러오기

## 실행 방법
1. 압축을 해제합니다.
2. `index.html`을 브라우저에서 열거나 저장소 루트에 업로드합니다.
3. Github Pages를 활성화하면 바로 실행할 수 있습니다.

## 비고
이 버전은 그래픽 기반 확장을 위한 시스템 원형입니다.
